# file-core
maven repository


## Description
* サービス上でファイルを扱うときに便利なクラス

## Install
* maven
```
<dependencies>
    <dependency>
        <groupId>miyakawalab.tool</groupId>
        <artifactId>file-core</artifactId>
        <version>${version}</version>
    </dependency>
</dependencies>
<repositories>
    <repository>
        <id>file-core</id>
        <url>https://raw.github.com/908ab/file-core/mvn-repo/</url>
    </repository>
</repositories>
```

## Version
> 1.0
